# 타입스크립트 백엔드를 쓰는 한국 회사는 어디가 있나요?

한국 회사 중 대부분은 자바백엔드를 사용하는데요.  
그렇기에 타입스크립트 백엔드를 사용하는 회사를 찾기가 힘들어 이렇게 모아봤습니다.  
만약 회사를 추가하고싶으시다면 [컨트리뷰팅 문서](./CONTRIBUTING.md)를 따라 PR 작성을 부탁드립니다.
